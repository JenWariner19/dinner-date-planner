How to Guide

  1. 1. Select how you would like to search for a meal (cuisine or protein).
  2. 2. Select a meal that you like to view a recipe for.
  3. 3. Select the type of liquor you would like to feature in your cocktail.
  4. 4. Select a cocktail that you like to view a recipe for.
  5. 5. Select "Save Pairing" if you want to save your choices, or you can start a new search.


        About

        The Dinner Date Planner helps you to plan a memorable dinner for that special someone. Search for a meal based on the type of cuisine or by the protein. Once you select the meal you would like to make search for a cocktail to serve as well. We will save your selected choices so that you can come back and view the pairings you have already done.
